# Wsb-zadanie1

Rozwiązania zadań programistycznych w językach Python i C#.

## Opis zadań

1. **Prosty kalkulator dwóch liczb**: Program prosi o dwie liczby i operację (+, -, *, /), oblicza wynik.

2. **Konwerter temperatur**: Konwersja między Celsjuszem a Fahrenheitem.

3. **Średnia ocen ucznia**: Oblicza średnią z ocen i sprawdza, czy uczeń zdał (średnia >= 3.0).

## Pliki

- `zadania.py`: Rozwiązania w Pythonie z menu wyboru.
- `zadania.cs`: Rozwiązania w C# z menu wyboru (switch).
- `Zadania.csproj`: Projekt dla C#.

## Uruchamianie

### Python
```bash
python3 zadania.py
```

### C#
```bash
dotnet run
```

## Branchy

- `main`: Wszystkie pliki.
- `python`: Tylko zadania.py.
- `csharp`: Tylko zadania.cs i csproj.
